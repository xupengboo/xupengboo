## 🖥️ 系统工具

### WinDirStat — C 盘清理好助手

**官网：** https://windirstat.net

**适用平台：** Windows。 

**核心功能：**

- **扫描 C 盘（或任意目录），用彩色方块热力图展示每个文件夹的占用比例，块越大占用越多**
- **点击色块直接定位到对应文件，支持右键删除，找到即清理**
- 按文件类型汇总占用排行，快速揪出 `.log`、`.tmp`、大型安装包等"空间杀手"

**下载安装：**

| 方式 | 命令 / 链接 |
|---|---|
| 官网下载（x64） | [WinDirStat-x64.msi](https://github.com/windirstat/windirstat/releases/latest/download/WinDirStat-x64.msi) |
| winget | `winget install -e --id WinDirStat.WinDirStat` |
| scoop | `scoop install extras/windirstat` |

:::tip 推荐使用 winget 安装
winget 是 Windows 官方包管理器，安装后可以用 `winget upgrade WinDirStat.WinDirStat` 一键升级，省去手动下载的麻烦。
:::
